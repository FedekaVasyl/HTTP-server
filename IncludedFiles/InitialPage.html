This is the starting page of my server

HTTP server written in C. C is a general-purpose, imperative computer programming language, supporting structured programming, lexical variable scope and recursion, while a static type system prevents many unintended operations.

Programming environment in which it is written server - Qt Creator. Qt is a cross-platform application framework that is widely used for developing application software that can be run on various software and hardware platforms with little or no change in the underlying codebase, while still being a native application with the capabilities and speed thereof.
About Server
The server is written using sockets technology. The server returns the requested page, responds to the mistakes from both the user and the server sides. The server keeps a log of events, because every event that happens on the server is written in a separate file (log.txt), which recorded the event status, time and date of occurrence and description of the event. Developed console menu for the service, which allows you to start and stop the server, enable and disable logging, change the path to the log-file, and also the correct exit from the program.
Clients
The clients are browsers on devices users. The user enters in the browser address http://127.0.0.1:8000 and comes to the homepage server.
